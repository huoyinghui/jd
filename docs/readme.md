## 电子商城管理端
* vue spa  

### 1.前端技术选择

* vue  
* vue-router
* element-ui
* axios
* Echars

#### 1.1 vue 全家桶
* 安装vue 脚手架
* 通过vue ui 创建项目
* 配置vue router
* 配置Element-UI 组件库
* 配置axio
* 安装开发依赖'less-loader'
* 安装开发依赖'less'
* 重启server

### 2.用户模块
* /api/login(username, password -> token)
* /api/logout


#### 2.1 登录布局(login form)
* el-from
* el-from-item
* el-input
* el-button
* 字体图标(用户名的图标/密码图标)


* session/cookie: 不跨域
* token: 跨域
